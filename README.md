# Task Management App - Frontend (Vue 3 + Vite)

Frontend untuk Task Management App

> Aplikasi Task Management ini dibuat oleh **Yudha Papua Setyo Atmaji**.

## Tech Stack

- Vue.js 3
- Vite
- Axios
- LocalStorage (JWT Token)

---

## Requirements

Pastikan sudah terinstall:

- Node.js 18+
- NPM / Yarn / PNPM

---

## Cara Menjalankan

1. Clone repository:

```bash
git clone https://github.com/username/task_management_app_frontend.git
```

2. Masuk ke folder task_management_app_frontend:

```bash
cd task_management_app_frontend
```

3. Install dependencies:

```bash
npm install
```

4. Jalankan aplikasi:

```bash
npm run dev
```

5. Aplikasi berjalan pada:

```bash
http://localhost:5173
```
